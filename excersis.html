QUESTION
    why should we use async await when we can fetch with promise

    --------------------------------------------------------------------
  1. They handle asynchronous operations (like fetching data)
  2. There are diffirence in how they are written and understood

  3. Promise works well but
    can be messy with lot of chaining (then inside then )
  4. Slightly harder to read/debug when deeply nested

  5. async/await
  6. Cleaner and easier to read>
  7. Feels like regular synchronous code
  8. Easier to handle complex logic and error handling
    9. lastly async/await is more preferable because, its more clear and readable code than Promise
